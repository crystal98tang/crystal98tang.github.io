# 1 Java的发展历史

C++为基础 GREEN平台 OAK平台

SUN（斯坦福大学网络） 09年被oracle收购

创始人James Gosling

JDK（Java Development Kit）

初始开发一共三类：

* Java SE：进行单机版程序开发，常作为基础平台

* Java ME：进行移动开发，没赶上好时代

    * 这项开发已经被Android取代

* Java EE：Java开发的企业版 服务器端

# 2 Java主要特点

#### Java应用程序运行机制：
    高级编程语言类型：++编译++、++解释++

Hello.java ------++编译++-----> Hello.class -----++解释++-----> Hello 
（Java虚拟机JVM for DOS、WinX、Unix）

#### Java特点：

  * 可移植性

  * 开发足够简单

  * 是为数不多**支持多线程**编程

  * 自动垃圾收集

  * 面向对象
